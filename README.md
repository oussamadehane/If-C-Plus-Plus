# If-C-Plus-Plus 🎛️

Welcome to **If-C-Plus-Plus**! This repository is dedicated to mastering **Conditional Logic** and **Control Flow** in C++. Here, I explore how to build intelligent, decision-based programs using C++ algorithms.

---

## 🛠️ Project List

| Project | Description |
| :--- | :--- |
| **Leap-Year-Validator** | Checks if a year is a leap year using complex logical operators. |
| **Voting-Eligibility-Checker** | Validates if a user can vote based on their age using conditional branching. |
| **Height-Classifier** | Classifies human height into categories (Dwarf, Short, Tall) using nested if-else. |
| **Largest-Number-Finder** | Efficiently identifies the maximum of three input integers. |
| **Even-Odd-Analyzer** | Determines number parity using the modulo operator. |
| **Equality-Tester** | Compares two integers for equality. |

---

## 🧠 Core Concepts Applied
* **Branching Logic**: Mastery of `if`, `else if`, and `else` statements.
* **Boolean Logic**: Utilizing `&&` (AND), `||` (OR), and `!` (NOT) operators.
* **Modular Design**: Separating logic, input, and output into distinct functions.
* **Data Validation**: Implementing `do-while` loops to ensure correct user input.
* **Enums**: Improving code readability with custom enumerations.
